## –°–∏–Ω—Ç–∞–∫—Å–∏—Å HDL

–Ø–∑—ã–∫ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —Ç–æ–∫–µ–Ω–æ–≤. –¢–æ–∫–µ–Ω—ã ‚Äî —ç—Ç–æ –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤ —è–∑—ã–∫–µ. –ü–æ-–ø—Ä–æ—Å—Ç–æ–º—É, —Ç–æ–∫–µ–Ω—ã –≤ —è–∑—ã–∫–µ ‚Äî —ç—Ç–æ –∫–∞–∫ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏ –ø—Ä–µ–ø–∏–Ω–∞–Ω–∏—è –≤ —Ç–µ–∫—Å—Ç–µ.

–ö–æ–º–ø—å—é—Ç–µ—Ä—É –Ω—É–∂–Ω—ã —Å—Ç—Ä–æ–≥–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–µ –ø—Ä–∞–≤–∏–ª–∞, —á—Ç–æ–±—ã –ø—Ä–æ—á–∏—Ç–∞—Ç—å –∫–æ–¥ –Ω–∞ —è–∑—ã–∫–µ HDL. –°–∏–Ω—Ç–∞–∫—Å–∏—Å —è–∑—ã–∫–∞ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –≤ –∫–∞–∫–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –º–æ–≥—É—Ç —Å—Ç–æ—è—Ç—å —Ç–æ–∫–µ–Ω—ã, —á—Ç–æ–±—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å –ø—Ä–∞–≤–∏–ª–∞–º –≥—Ä–∞–º–º–∞—Ç–∏–∫–∏ —è–∑—ã–∫–∞. –¢–æ–∫–µ–Ω—ã –Ω–µ –º–æ–≥—É—Ç —Å—Ç–æ—è—Ç—å –≤ –ª—é–±–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏, —Ç–∞–∫–∂–µ –∫–∞–∫ –∏ —Å–ª–æ–≤–∞ –≤ —ç—Ç–æ–º —Ç–µ–∫—Å—Ç–µ.

–ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —è–∑—ã–∫–µ HDL –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —á–µ—Ç—ã—Ä–µ –∫–ª—é—á–µ–≤—ã—Ö —Å–ª–æ–≤–∞: `CHIP`, `IN`, `OUT`, `PARTS`. –ò –¥–≤–µ–Ω–∞–¥—Ü–∞—Ç—å –∑–Ω–∞–∫–æ–≤: `{ . ( : / [ } ] * = ) ,`. –í—Å–µ–≥–æ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç—å —Ç–æ–∫–µ–Ω–æ–≤.

–ö–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ –∏ –∑–Ω–∞–∫–∏ –æ–±—Ä–∞–∑—É—é—Ç –≤–æ—Ç —Ç–∞–∫—É—é —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫—É—é –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—é –æ–ø–∏—Å–∞–Ω–∏—è —ç–ª–µ–∫—Ç—Ä–æ—Å—Ö–µ–º—ã:

–ö–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ PARTS –∏ –∑–Ω–∞–∫ : (PARTS:) –æ–∑–Ω–∞—á–∞—é—Ç –Ω–∞—á–∞–ª–æ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å—Ö–µ–º—ã.

–î–∞–≤–∞–π—Ç–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –æ–ø–∏—Å–∞–Ω–∏—è —Ä–∞–∑–Ω—ã—Ö –≥–µ–π—Ç–æ–≤ –Ω–∞ HDL.
